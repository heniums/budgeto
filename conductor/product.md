# Product Guide: Budgeto

## Vision
Budgeto is a personal finance and budgeting application that helps individuals take control of their money. It enables users to track expenses, manage budgets, and gain financial insights through clear reporting.

## Target Users
- **Primary:** Individuals managing personal day-to-day spending.
- Secondary consideration for future: families and freelancers.

## Core Goals
1. **Track Expenses** — Log and categorize transactions as they happen.
2. **Manage Budgets** — Define budgets per category and monitor progress.
3. **Financial Insights** — Surface trends and summaries via reports and dashboards.

## Key Features
- **Expense Categorization** — Tag and categorize spending (automatic suggestions + manual override).
- **Reports & Dashboards** — Visual summaries (monthly/yearly) of income, spending, and budget health.
- *(Foundational)* Expense entry and budget definition underpin the above.

## Platform
- **Progressive Web App (PWA)** — A responsive web application that also installs on Android, allowing use on both web and mobile from a single codebase.

## Success Criteria
- A user can record an expense, assign a category, and view it reflected in a dashboard within one session.
- A user can define a monthly budget per category and see progress against it.
- The app is installable on Android and usable offline for core views (PWA).
